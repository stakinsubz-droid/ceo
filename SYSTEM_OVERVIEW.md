## 💡 HOW TO USE

### 1. Start Backend
```bash
cd /workspaces/ceo
uvicorn backend.server:app --reload
```

### 2. Test LEVEL 1 (Basic)
```bash
curl -X POST http://localhost:8000/api/run
```

### 3. Test LEVEL 2 (Monetization)
```bash
# Configure scaling
curl -X POST "http://localhost:8000/api/v2/scale/configure?projects_per_day=5"

# Generate social content
curl -X POST "http://localhost:8000/api/v2/social/generate/{product_id}?days=180"

# Get revenue metrics
curl http://localhost:8000/api/v2/revenue/all-metrics
```

### 4. Test LEVEL 3 (Enterprise)
```bash
# Enable enterprise scaling
curl -X POST "http://localhost:8000/api/v3/enterprise/enable-scaling?projects_per_day=50"

# Generate YouTube shorts
curl -X POST "http://localhost:8000/api/v3/youtube/generate-shorts/{product_id}?days=30"

# Create email sequences
curl -X POST "http://localhost:8000/api/v3/email/generate-sequence/{product_id}?days=30"

# Start support bot
curl -X POST "http://localhost:8000/api/v3/support/create-bot/{product_id}"

# Run 50 projects parallel
curl -X POST http://localhost:8000/api/v3/enterprise/run-cycle/50
```
