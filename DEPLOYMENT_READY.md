## 🔑 WHAT YOU NEED TO SET UP

### On Render Dashboard
Add these environment variables:
1. MONGO_URL (required)
2. DB_NAME (required)
3. GUMROAD_TOKEN (optional, for GumRoad)
4. YOUTUBE_API_KEY (optional, for YouTube)
5. MAILCHIMP_API_KEY (optional, for email)
6. OPENAI_API_KEY (optional, for support bot)
7. STRIPE_API_KEY (optional, for payments)
(+ 10+ more optional platform tokens)

### On Vercel Dashboard
Add these environment variables:
1. REACT_APP_API_URL = your-render-backend-url
2. REACT_APP_ENABLE_LEVEL3 = true

That's it!
